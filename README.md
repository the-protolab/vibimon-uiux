# Prototipo Game Boy UI1/UI2


The goal of this prototype is to compare two navigation approaches and evaluate their impact on information discoverability, cognitive load and player flow.

This prototype is the implementation of a prior UX exploration conducted in Figma, where different hypotheses and trade-offs were evaluated.
